# Charter — Go Course: Robusto y Práctico

## Problem Statement
El curso actual de Go tiene 12 módulos con contenido teórico y componentes interactivos básicos (Quiz, DragDrop, BranchingScenario, Playground). Sin embargo, carece de profundidad pedagógica, ejercicios prácticos progresivos, y no refleja el estado actual del lenguaje Go (1.21+).

## Goal
Transformar el curso en el recurso más completo y práctico de Go en español, basado en evidencia pedagógica (learning science) y alineado con Go 1.22+ y las mejores prácticas actuales de la industria.

## Success Criteria
- Cada módulo incluye al menos un ejercicio práctico con código real ejecutable
- El contenido cubre Go 1.21+ features (generics avanzados, slices/maps stdlib, rangeover funcs)
- Los quizzes aplican espaciado de repetición y retrieval practice
- Cada módulo tiene ejercicios de dificultad progresiva (básico → intermedio → desafío)
- El curso incluye proyectos integradores que conectan múltiples módulos
- Las explicaciones usan analogías concretas antes del código abstracto

## Scope
### In Scope
- Actualizar contenido de los 12 módulos existentes con material más profundo y práctico
- Agregar ejercicios de código progresivos por módulo
- Mejorar componentes interactivos con feedback más rico
- Añadir mini-proyectos integradores (al menos 3)
- Actualizar a Go 1.22+ (range over integers, slices package, maps package, structured logging)
- Aplicar principios de Cognitive Load Theory y Retrieval Practice en el diseño instruccional

### Out of Scope
- Backend / servidor de ejecución propio (sigue usando playground externo)
- Sistema de usuarios o auth
- Soporte multi-idioma en esta iteración

## Stakeholders
- Estudiantes de Go hispanohablantes (desde junior developers hasta devs con experiencia en otros lenguajes)
- Autor del curso

## Timeline
- Investigación y diseño instruccional: fase actual
- Implementación por módulo: iterativa post-diseño
